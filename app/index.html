Сантехника «Кит»

Разработка интернет-магазина сантехники на редакции 1С-Битрикс24: Интернет-магазин + CRM

Интернет-магазин «Кит» - магазин с тонко подобранным ассортиментом товара, как по техническим и дизайнерским характеристикам, так и по соотношению цена-качество.

Цель разработки сайта

Увеличить прибыль компании за счет роста количества онлайн-продаж сантехники розничным покупателям и автоматизации обработки заказов.

Задачи

  • Упростить процесс обновления цен и остатков товара на сайте с помощью настройки выгрузки товаров на сайт из файлов Excel с помощью модуля импорта

  • Организовать удобный поиск и подбор товаров на сайте, настроить умный фильтр по свойствам для каждой категории каталога

  • Автоматизировать процесс оформления заказа на сайте для клиентов - настроить способы доставки по весу товара и подключить онлайн-оплату

  • Автоматизировать процесс обработки заказа для менеджера интернет-магазина

  • Настроить этапы обработки заказов в Битрикс24
